# Bash Configuration

This directory contains Bash shell configuration files.

## Files

- `.bashrc` - Main bash configuration file for interactive shells

## Installation

To use this configuration:

```bash
# Backup your existing .bashrc
cp ~/.bashrc ~/.bashrc.backup

# Copy the new configuration
cp bash/.bashrc ~/.bashrc

# Reload the configuration
source ~/.bashrc
```

## Customization

Feel free to modify the following sections:
- **Aliases**: Add your custom command shortcuts
- **Prompt**: Customize the PS1 variable for your preferred prompt style
- **History**: Adjust HISTSIZE and HISTFILESIZE for your needs
