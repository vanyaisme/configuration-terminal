# Utility Scripts

This directory contains utility scripts for managing terminal configurations.

## Scripts

### backup.sh
Backs up your current terminal configurations to a timestamped directory.

**Usage:**
```bash
chmod +x scripts/backup.sh
./scripts/backup.sh
```

This will create a backup directory in your home folder with the current date and time, containing all your existing configuration files.
